# Archiwum dokumentacji deweloperskiej

Ten folder zawiera historyczne notatki deweloperskie i dokumenty tymczasowe z procesu rozwoju projektu PrettyScreenSHOT.

## Zawartość

### Naprawione problemy
- `NAPRAWIONE_PROBLEMY.md` - Lista naprawionych problemów kompilacji (wersja 1)
- `NAPRAWIONE_PROBLEMY_V2.md` - Lista naprawionych problemów kompilacji (wersja 2)
- `NAPRAWIONE_NIEJEDNOZNACZNOSCI.md` - Naprawione niejednoznaczności w kodzie

- `AUTO_UPDATE_PLAN.md` - Plan implementacji systemu automatycznych aktualizacji
- `GUI_REDESIGN_PLAN.md` - Plan przeprojektowania GUI
- `NEUMORPHIC_CONVERSION_GUIDE.md` - Przewodnik konwersji do stylu neumorficznego
