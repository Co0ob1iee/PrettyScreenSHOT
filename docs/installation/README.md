# Dokumentacja instalacji PrettyScreenSHOT

Ten folder zawiera dokumentację związaną z instalacją i dystrybucją aplikacji PrettyScreenSHOT.

## Pliki

- **INSTALLATION.md** - Główny przewodnik instalacji dla użytkowników końcowych
- **INSTALLER_GUIDE.md** - Przewodnik tworzenia instalatorów dla deweloperów
- **README_INSTALLER.md** - Dodatkowe informacje o procesie instalacji

## Dla użytkowników końcowych

Jeśli chcesz zainstalować PrettyScreenSHOT, zacznij od [INSTALLATION.md](INSTALLATION.md).

## Dla deweloperów

Jeśli tworzysz instalator lub dystrybucję aplikacji, zobacz [INSTALLER_GUIDE.md](INSTALLER_GUIDE.md).
